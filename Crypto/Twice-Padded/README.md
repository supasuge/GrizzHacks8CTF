# Twice Padded
- **Author:** Evan Pardon | [supasuge](https://github.com/supasuge)
- **Category:** Crypto
- **Difficulty:** Easy
- **Flag Format:** `Grizz{...}`

## Description

A junior engineer encrypted two internal messages using a "perfectly secure" One-Time-Pad that he assures us will never be broken. He followed the rules carefully, except for one small detail, can you spot it and decrypt the flag given the partial plaintext?
